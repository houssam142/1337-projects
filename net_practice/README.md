# Description
  NetPractice is a project designed to introduce the basics of IPv4 networking.It's goal is to help you understand IP addresses, subnet masks, and routers work by fixing broken network configurations.

# Instructions
  to get started with the exercises, you need:
  • install the compressed file net_practice.1.7.tgz
  • use the command line 'tar -xvf' to decompress the file
  • then cd net_practice, and 'open index.html' to open the project interface in a browser of your choice
